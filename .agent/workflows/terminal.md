## Terminal Environment

This project uses **WSL (Windows Subsystem for Linux)** as the default terminal, NOT PowerShell.

### Command Syntax
- Use Linux/bash commands (ls, cat, grep, etc.)
- Do NOT use PowerShell cmdlets (Get-ChildItem, Get-Content, etc.)
- File paths work with both Windows and Linux style paths

### NPM Commands
// turbo-all
```bash
npm install
npm run dev
npm run build
```
